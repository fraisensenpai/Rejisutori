# Rejisutori

AI-powered habit, prayer, and life management app with voice assistant Tenka.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Lucide React
- **Backend**: Supabase (PostgreSQL), Edge Functions (Deno)
- **AI**: Google Gemini API (Direct integration)

## Features
- AI-driven life companion "Tenka"
- Habit and prayer tracking
- Structured task management
- Daily reflections and mood logging
